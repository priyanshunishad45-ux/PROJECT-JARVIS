# Jarvis Voice Assistant (Python)

A simple voice-controlled assistant built using Python.  
It can:

- Open Google, Facebook, YouTube, LinkedIn
- Play songs from a custom music library
- Fetch and speak latest news headlines (India)
- Respond to the wake word: "Jarvis"
